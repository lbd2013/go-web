## 拉取代码
```
先生成 Personal access tokens
然后用 git clone https://lbd2013:<Personal access tokens>@github.com/lbd2013/当前项目名 进行拉取
```

## 初始化
```
查看cmd命令：
go run main.go -h

创建表：
go run main.go migrate up

```

## 启动方法
```
开两个窗口
1. air (自动编译运行)
2. Mailhog (邮件服务器，用于注册)
```
